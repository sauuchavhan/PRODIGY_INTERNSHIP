# Prodigy Infotech Cybersecurity Internship

This repository contains all tasks completed as part of the Cybersecurity Internship at Prodigy Infotech.

## Tasks

### Task 01 - Caesar Cipher
Text encryption and decryption using Caesar Cipher.

### Task 02 - Image Encryption Tool
Image encryption and decryption using pixel manipulation.

### Task 03 - Password Complexity Checker
Password strength analysis based on security parameters.

### Task 04 - Simple Keylogger
Keystroke logging application for educational purposes.

### Task 05 - Network Packet Analyzer
Network packet capture and protocol analysis using Scapy.

## Technologies Used
- Python
- Pillow
- Scapy
- Pynput
- Regular Expressions
