# Task 01 - Caesar Cipher

## Description
This project implements the Caesar Cipher algorithm to encrypt and decrypt text messages. The user provides a message and a shift value, and the program performs encryption and decryption accordingly.

## Features
- Encrypt text using a shift value
- Decrypt encrypted text
- Supports uppercase and lowercase letters
- User-friendly console interface

## Technologies Used
- Python

## How to Run

```bash
python caesar_cipher.py
