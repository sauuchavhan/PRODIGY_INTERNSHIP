# Task 02 - Image Encryption Tool

## Description
This project performs image encryption and decryption using pixel manipulation techniques. The program modifies RGB pixel values using a secret key and restores them during decryption.

## Features
- Encrypt images using a key
- Decrypt encrypted images
- Pixel-level image manipulation
- Simple and effective image security demonstration

## Technologies Used
- Python
- Pillow Library
