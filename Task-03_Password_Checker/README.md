# Task 03 - Password Complexity Checker

## Description
This project evaluates password strength based on security parameters such as length, uppercase letters, lowercase letters, numbers, and special characters.

## Features
- Checks password strength
- Categorizes passwords as:
  - Weak
  - Medium
  - Strong
- Provides instant feedback

## Technologies Used
- Python
- Regular Expressions (re)

## How to Run

```bash
python password_checker.py
```

## Example

Input:
Abc@12345

Output:
Password Strength: Strong
