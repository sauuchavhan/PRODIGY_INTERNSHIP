# Task 04 - Simple Keylogger

## Description
This project records keyboard keystrokes and stores them in a text file for educational and authorized testing purposes.

## Features
- Captures keyboard input
- Logs keystrokes into a file
- Supports normal and special keys

## Technologies Used
- Python
- Pynput Library

## Installation

```bash
pip install pynput
```

## How to Run

```bash
python keylogger.py
```

## Output
The captured keystrokes are saved in:

```text
keylog.txt
```

## Disclaimer
This project is developed strictly for educational purposes and should only be used with proper authorization.
