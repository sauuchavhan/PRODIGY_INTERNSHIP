# Task 05 - Network Packet Analyzer

## Description
This project captures and analyzes network packets using the Scapy library. It displays information such as source IP address, destination IP address, and protocol details.

## Features
- Packet sniffing
- Source IP detection
- Destination IP detection
- Protocol analysis
- Real-time packet monitoring

## Technologies Used
- Python
- Scapy

## Installation

```bash
pip install scapy
```

## How to Run

```bash
python packet_analyzer.py
```

## Example Output

Source IP: 192.168.1.5
Destination IP: 8.8.8.8
Protocol: IP / TCP
